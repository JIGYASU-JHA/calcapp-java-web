# CalcApp Java Web Application

A simple calculator web application developed using Java Servlets, HTML, and Apache Tomcat.

## Features
- Addition
- Subtraction
- Multiplication
- Division

## Technologies Used
- Java
- Servlet API
- HTML5
- Apache Tomcat

## Project Structure

WEB-INF/
 ├── classes/
 │    └── CalcSrv.java
 └── web.xml
 |__ index.html

## How to Run
1. Deploy project inside Apache Tomcat webapps folder
2. Start Apache Tomcat server
3. Open localhost in browser
